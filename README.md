Fui desde lo más básico a lo más complejo. En un primer lugar, indiqué que se guardasen los prompts para no tener que hacerlo a mano. Después indiqué el rol que quería que tuviese la IA, entendiendo que de esta manera, no habría que especificar exactamente todo lo que quería, ya que al tener un rol senior, estarán implícitas.

Luego le di la estructura del proyecto y le dije la funcionalidad necesaria, así como los requisitos técnicos, mejoré los estilos y refactoricé la clase task, sacándola a parte para separar responsabilidades y poder testearla, además de hacer app,js más legible.

Por último, pedí que se crease el README.md, para poder escribir el resumen de lo indicado.
